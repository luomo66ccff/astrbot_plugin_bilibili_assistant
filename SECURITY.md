# 安全说明

本插件会处理 B站 Cookie、评论内容和回复草稿。请在部署时注意以下事项：

- 不要把 `bilibili_cookie` 写入仓库、截图或公开日志。
- 建议使用专门的运营账号，不要使用主账号直接开启自动化能力。
- 首次部署保持 `dry_run=true` 和 `require_confirmation=true`。
- 开启自动回复前，请先确认黑名单、敏感词、频率限制和人工审核流程。
- 如果 B站返回风控错误，应降低请求频率、检查登录状态或更换更稳定的出口网络。

如果发现安全问题，请通过 GitHub Issue 描述复现方式、影响范围和建议修复方案。请不要在 Issue 中粘贴真实 Cookie、账号凭证或隐私数据。
